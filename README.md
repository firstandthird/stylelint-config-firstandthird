Stylelint rules for First+Third frontend.

## Installation

```sh
npm install stylelint-config-firstandthird
```

_or_

```sh
yarn add stylelint-config-firstandthird
```

## Usage

```json
// package.json
{
  "devDependencies": {
    "stylelint": "^13.10.0",
    "stylelint-config-firstandthird": "^1.2.0"
  },
  "stylelint": {
    "extends": "stylelint-config-firstandthird"
  }
}
```
